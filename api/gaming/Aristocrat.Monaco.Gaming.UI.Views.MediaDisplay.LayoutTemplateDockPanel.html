    Class LayoutTemplateDockPanel

    This class is intended to maintain the different screen areas per Player User Interface Protocol V1. The service window and banner areas will have content set by other systems. The main content should be directly inline in the XAML. The logic to set and maintain sizes are controlled in the LayoutTemplateViewModel.
