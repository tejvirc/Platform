    Class ProgressiveLevel

    The fundamental data structure for a progressive level defined by a game. IProgressiveLevelProvider will manage all progressive levels for all games.

    Turnover

    Gets or sets a turnover value. Turnover games, also called hyperlink games, can enable the game to use a random number between 0 and the turnover value to determine if a progressive level can be hit. The platform doesn't care about this value because the game will always perform this calculation. Turnover games can be both linked and standalone.
