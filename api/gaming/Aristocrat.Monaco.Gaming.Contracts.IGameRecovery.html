    AbortRecovery()

    Aborts a recovery. This is called if the platform has a legitimate reason to interrupt a game in the middle of recovery. An example of this is if a lockup is generated during recovery, and the use does a replay--the replay will need to abort the recovery. An abort should not count as a recovery retry since it didn't fail due to error. When possible, the platform should restart the recovery.
