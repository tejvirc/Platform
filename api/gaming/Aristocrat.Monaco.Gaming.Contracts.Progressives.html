    IProtocolLinkedProgressiveAdapter

    Provides an adapter interface for the protocol layer to communicate with progressives in order to prevent API calls from protocols not configured to handle progressives. Filters out all API calls made to ILinkedProgressiveProvider and IProgressiveConfigurationProvider.

    Most API calls require the protocolName to be included which be used for filtering. Any protocol can view progressive data which is why view calls don't have a progressiveName parameter.
