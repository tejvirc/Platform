    Class QueueTransactionRequestorProxy

    Used to serve as an ITransactionRequestor proxy to any transaction requestor that wants to have the TransactionCoordinator block for them. This allows a requestor to not worry about implementing the ITransactionRequestor interface or the necessary steps to make it happen and instead just get a transaction Guid.
