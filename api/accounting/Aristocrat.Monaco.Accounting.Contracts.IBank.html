    Interface IBank

    Provides an interface to interact with the Bank.

    Namespace: Aristocrat.Monaco.Accounting.Contracts
    Assembly: Monaco.Accounting.Contracts.dll
    Syntax
    public interface IBank
    Remarks

    The service which implements this interface handles deposits, withdrawals, and querying of credits, also known as AccountTypes in the XSpin.

    When credits are changed within the bank a BankBalanceChangedEvent will be emitted. The service which implements this interface already has the protections in place to limit the access to credits outside of the currently processing transaction and prevent the credit overflow.

    The amounts that the bank deals with are expected to be in milli-units such that 1000 millicents is equivalent to 1.

    Deposit(AccountType, Int64, Guid)

    Method to deposit to the specified account with the amount after checking if the provided transactionId is the active transaction.

    Declaration

    Parameters
    AccountType account

    Type of account to deposit into.
    Int64 amount

    Amount to deposit.
    Guid transactionId

    Guid of the transaction.
    Examples
      long amount = 50000;
  Guid transactionId = Guid.NewGuid();
  ITransactionCoordinator coordinator = ...;
  if (_persistedTransaction.TransactionGuid == Guid.Empty)
  {
    // Transaction coordinator denied the transaction
    return;
  }

  IBank bank = ...;
  if (bank.CheckDeposit(AccountType.Cashable, amount, transactionId))
  {
    // Bank authorized, depositing
    bank.Deposit(AccountType.Cashable, amount, transactionId);
  }
  else
  {
    // Bank denied the deposit, ending the transaction
  }

  coordinator.ReleaseTransaction(transactionId);
  ResetState();
