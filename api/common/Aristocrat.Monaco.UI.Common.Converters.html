    BoolToVisibilityConverter

    Converts bool to visibility property. By setting the TrueValue and False value properties, the default behavior can be customized. In particular, you can use this to invert the default behavior to make something collapsed when true. Xaml usage: converters:BoolToVisibilityConverter x:Key="BoolToCollapsed" TrueValue="Collapsed" FalseValue="Visible" Grid x:Name="GameTopScreenRoot" Visibility="{Binding IsLobbyVisible, Converter={StaticResource BoolToCollapsed}}"
