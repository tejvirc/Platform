    OperatorMenuDialogWindow

    OperatorMenuDialogWindow is a full-screen transparent wrapper for our modal dialogs to get around a WPF touch bug. As of August 2019, with WPF, if you have a modal dialog up via ShowDialog and the user touches repeatedly outside of the modal dialog, the touch interface will lock up, sometimes for up to 30 seconds By making our dialogs full-screen but transparent (.01 opacity) we can capture all of the clicks "outside" of the modal dialog on the same window as the actual dialog, thus working around the WPF bug.
