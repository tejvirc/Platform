    Class AdaptToOrientationGridAction

    Resizes the first row of the Grid to 0 if the containing Window has a Landscape aspect-ratio. Otherwise, the Row height is set to {FirstRowHeightWhenPortrait} when Window has a Portrait aspect-ratio.
