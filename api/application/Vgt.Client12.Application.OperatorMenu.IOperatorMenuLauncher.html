    Interface IOperatorMenuLauncher

    An interface to expose the status of the operator launcher and commands which can be used to send requests to the launcher.

    Namespace: Vgt.Client12.Application.OperatorMenu
    Assembly: Monaco.Application.Contracts.dll
    Syntax
    public interface IOperatorMenuLauncher
    Remarks

    This is a service interface. The main duty of service implementing this interface is to manage the operator menu like showing or hiding it, and to report the operator menu's status, e.g, whether it is showing up.

    EnableKey(Guid)

    Instructs the implementation to begin/resume operator key usage.

    Declaration

    Parameters
    Guid enabler

    The object enabling the functionality.
    Remarks

    This method sends a request to the launcher to make the operator key work for the enabler. But The operator key won't work until no DisableKey request is pending.

    Hide()

    Hides the operator menu. The main motivation for this is replay which needs to temporarily hide the operator menu to show the replay, but then return to the operator menu. We do not want to close the operator menu in replay, but keeping the window open is causing windowing order bugs.
