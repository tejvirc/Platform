    Class CompositeMeter

    A meter implementation for meters in which all values in the different time frames are calculated from a formula.

    public class CompositeMeter : IMeter, IDisposable
    Remarks

    All values in three time frames are not persisted in the storage because they are calculated at the runtime.

    The Fast Lightweight Expression Evaluator (Flee) is used to evaluate an expression. You can download it from http://flee.codeplex.com/releases/view/7153 to learn more.
