    Interface ITime

    An interface to provide the functionalities related to the date and time updating, formatting, and converting for display or other purposes.

    Namespace: Aristocrat.Monaco.Application.Contracts
    Assembly: Monaco.Application.Contracts.dll
    Syntax
    public interface ITime
    Remarks

    Presently the parameter of type DateTime in the methods of this interface must be in UTC. The OS clock will always be set to UTC. This interface should be used to convert UTC times to local times for display purpose.

    It is supposed that the component implementing this interface will post TimeUpdatedEvent when the date time is updated successfully.

    GetLocationTime()

    This method converts the DateTime into a location specific time with applied time zone adjustments.

    Declaration
    DateTime GetLocationTime()
    Returns
    DateTime

    A DateTime that has been converted to local time.
    Remarks

    In order to convert the supplied time to the wanted time, the used time zone must be set. Otherwise, it will just return the time supplied through the parameter.

    Examples
     // Get the time service.
 ITime time = ...;

 // Convert the local time to the time for the currently used
 // time zone.
 string timeString = time.GetLocationTime(DateTime.UtcNow).ToString("G",
                     CultureInfo.CurrentCulture);
