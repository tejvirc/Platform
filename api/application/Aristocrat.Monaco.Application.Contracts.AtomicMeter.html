    A meter implementation for meters without any compound formulas.

    Syntax
    public class AtomicMeter : IMeter, IReadWriteLockable, ILockable, IDisposable
    Remarks

    A meter instantiated with this class does not depend on any other meters to get its value. The Lifetime and Period values are persisted.

    public void ResetMeter(long resetValue)
    Parameters
    Int64 resetValue

    The value to reset meter values.
    Remarks

    In this method, all meter values in three time frames will be reset to the specified value. The meter value's wrap-around in terms of the classification.

    The change is notified if the MeterChangedEvent is hooked up.
