    StartEventHandler()

    Subscribes to events and kicks off the UI refresh timer. The OnRefreshTimeout method, which is called each time the timer elapses, will dispatch a call to UpdateScreen, which processes events retrieved from the event bus. UpdateScreen cannot run concurrently with StartEventHandler or StopEventHandler to prevent the scenario where UpdateScreen tries to get events and this object is not subscribed to any events.
