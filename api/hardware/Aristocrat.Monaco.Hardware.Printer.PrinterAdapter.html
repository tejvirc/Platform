    IPrinter.Print(Ticket)
    Remarks

    A ticket is an object defined and obtained from the TicketContent component. A ticket object is created by a user, all of the require information is applied to it, and then the user prints the ticket with this method. This method call the TicketContent's Render method, which converts the ticket to a stream of characters that will print a correctly formatted form.
