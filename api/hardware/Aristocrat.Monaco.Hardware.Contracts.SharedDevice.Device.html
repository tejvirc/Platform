    Parameters
    String protocol

    Protocol used by the device.
    String mode

    Mode of communication used (e.g., RS232, USB).
    String portName

    Communication port to which the device is connected.
    Int32 baudRate

    Baud rate at which the device is connected. The baud rate is the number of times per second a serial communication signal changes states; a state being either a voltage level, a frequency, or a frequency phase angle.
    Parity parity

    Parity is an optional parameter used in serial communications to determine if the data character being transmitted is correctly received by the remote device.
    Int32 dataBits

    Data bits indicates the number of bits used to represent a single data character during serial communication.
    StopBits stopBits

    Stop bits are used in asynchronous communication as a means of timing or synchronizing the data characters being transmitted.
    Handshake handshake

    Handshake is used to describe the method in which a serial device controls the amount of data being transmitted to itself.
