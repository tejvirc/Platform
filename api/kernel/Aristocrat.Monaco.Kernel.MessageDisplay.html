    IMessageDisplay.RemoveMessage(DisplayableMessage)
    Remarks

    Use this method to remove a message displayed (or to be displayed) by the active handler. The object passed in as a parameter is used to locate the message to be removed from the service and the message handler. An exception is NOT thrown if a matching object is not found. As of this writing, object identity determines equality, so the object reference used in the RemoveMessage call must be the same as the one sent to DisplayMessage

    IMessageDisplayHandler handler

    The object to no longer use as a message handler.
    Implements
    IMessageDisplay.RemoveMessageDisplayHandler(IMessageDisplayHandler)
    Remarks

    Sometimes a message display handler needs to be removed when a component that provides the message display is stopping or going out of context. For example, a game providing the message display capability goes away if the operator enters the operator menu. In this case the game message display component would need to remove itself as a message handler.
