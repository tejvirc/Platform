    Interface IMessageDisplayHandler

    Definition of the IMessageDisplayHandler interface.

    Namespace: Aristocrat.Monaco.Kernel
    Assembly: Aristocrat.Monaco.Kernel.Contracts.dll
    Syntax
    public interface IMessageDisplayHandler
    Remarks

    Calls to methods in this interface are made from an object that provides services accessible via the IMessageDisplay interface. A class that implements IMessageDisplayHandler should register itself with the IMessageDisplay service. It will then receive calls with messages from that service. The implementation of a message display handler component determines how messages are presented to the outside world.
