    Remarks

    It is important that all subscribers cancel their event subscriptions when they are no longer needed. Subscribers can call Unsubscribe() for each subscribed event type, or call UnsubscribeAll(). Subscribers should expect that their event queue will be destroyed once all subscriptions are canceled, even if the queue is not empty.
