    Class ApplicationMessage

    In ASP5000 all commands and responses are in the form of packets. An odd number of bytes is allowed in a packet at the application level. The general formats of messages from the MCI are as follows: Command = Set_Parameter (0x01) Class Type Parameter Number Data 1 byte 1 byte 1 byte 1 byte n bytes Command = Get_Parameter (0x02) Class Type Parameter Number 1 byte 1 byte 1 byte 1 byte Command = Set_Event (0x04) Class Type Parameter Number 1 byte 1 byte 1 byte 1 byte Command = Clr_Event (0x05) Class Type Parameter Number 1 byte 1 byte 1 byte 1 byte
