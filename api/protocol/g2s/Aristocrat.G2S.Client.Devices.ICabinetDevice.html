    Evaluate()

    Evaluates the device to determine if the device can be enabled. If all of the faults for a device have been cleared then the Enabled attribute for the device can be set to true. If any one fault still exists then the egmEnabled attribute MUST be set to false. This will also evaluate the State attribute. The lock and state may be escalated as a result of the evaluation.

    Declaration
    bool Evaluate()
    Returns
    Boolean

    true, if the state changed
