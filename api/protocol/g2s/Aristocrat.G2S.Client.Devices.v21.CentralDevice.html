    Class CentralDevice

    The central class is used to manage the distribution of finite chances between a central determinant game host and an EGM.The central class includes commands and events to request and issue finite chances and to associate EGM game combos and wager categories with finite deals supported by the central host.The central class also includes commands to retrieve the central transaction log maintained by the EGM.
