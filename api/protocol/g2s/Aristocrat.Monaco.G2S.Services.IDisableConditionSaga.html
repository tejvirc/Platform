    The disable condition
    TimeSpan timeToLive

    Func<String> message

    The message to display while the EGM is disabled
    Boolean persist

    true if the state should be persisted. If it's persisted the state will be recovered when calling Reenter()
    Action<Boolean> onDisable

    The callback used indicate whether or not the system was disabled. True if disabled.
    Remarks

    Persisting the state does not apply to queued requests. It's only applied for requests that obtain a lock. This should only be used for very specific cases, like the host explicitly locking the cabinet (EnterOptionConfig/EnterCommConfig)
