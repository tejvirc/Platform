    Class VendorSpecificInformation

    Information provided by DHCP server. GSA vendor-specific information provides network location, protocol to be used, and configuration parameters of a G2S and/or S2S communications configuration server, and servers involved in certificate management.It is expected that the location and required parameters of other GSA servers (device owners) will be provided by a communications configuration server via methods defined in the G2S and/or S2S protocol.
